#navigation and file management
pwd - print working directory
  : show you the directoy you are in .
cd - change directory
  : takes you to the dirctory you want to go to.
ls - list 
  : shows a list of the file and directories you want to see.
mkdir - make directoy  
  : creates new directory
touch - ...
  : creates new files
cp - copy
  : copies files and directories
mv - move
  : move files and directories and/or renames them .
rm - remove 
  : removes files or directories.
